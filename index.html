Weather Analysis: Summary


The objective of this project was to analyze how the weather changes as you approach the equator. To complete this analysis, weather data of 500 random cities was obtained using the OpenWeatherMap API. Matplotlib was used to plot the latitude vs. maximum temperature, humidity, cloudiness, and wind speed. The visualizations created and source data are presented on this site along with a description of the observed trends.
